# Young Ah Shin's Website

This version uses a simple academic homepage layout inspired by traditional university faculty pages.

## Files

- `index.html`: Home
- `about.html`: Biography, education, teaching, and international engagement
- `research.html`: Research interests and publications
- `projects.html`: Research projects, patents, and software
- `styles.css`: Shared design
- `cvShin_updated.pdf`: CV

## Upload to GitHub Pages

Upload every file directly to the root of:

`youngahshin/youngahshin.github.io`

The repository should look like:

```text
index.html
about.html
research.html
projects.html
styles.css
cvShin_updated.pdf
README.md
```

Do not upload the outer folder itself.

## Add a profile image

1. Upload a square or portrait image named `profile.jpg`.
2. Open `index.html`.
3. Replace `YS` inside `<div class="profile-image">` with:

```html
<img src="profile.jpg" alt="Young-A Shin">
```
